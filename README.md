## Goal
To explore the design and security trade-offs of **Decentralized Identity Tokens (DIT)**, inspired by:
- TNT (Trust Network Tokens)
- VouchMe (On-chain testimonials)

## Scope (Intentionally Small)
- ERC-721 based self-issued identity tokens
- Optional identity metadata (name, links, age group, etc.)
- On-chain endorsements between identity tokens
- Revocable endorsements
- Basic compromise / revocation signals (exploratory)

## Non-Goals
- Full DID standard compliance
- Off-chain attestations
- ZK / privacy guarantees (future work)
